Weather App Project

I created a live weather web application using HTML, CSS, and JavaScript, powered by a real-time weather API. The app dynamically displays the current weather conditions and a seven day forecast, updating automatically based on live data from the API. To enhance the user experience, the background of the application changes according to the weather for example, showing a sunny sky, clouds, or rain animations that match the live conditions. This project demonstrates my ability to integrate APIs, handle asynchronous JavaScript functions, and design a responsive, visually engaging interface that delivers both functionality and aesthetic appeal.
